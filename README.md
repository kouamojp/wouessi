# Wouessi Digital

Wouessi Digital est une plateforme web moderne développée pour une agence numérique spécialisée dans l'ingénierie logicielle et le marketing digital. Ce projet est une application React haute performance, conçue pour présenter les services, le portfolio et l'expertise technique de l'agence.

## 🚀 Fonctionnalités principales

- **Services d'Ingénierie Logicielle :** Présentation d'expertises variées (Web, Mobile, Cloud).
- **Marketing Digital :** Sections dédiées au SEO (Mobile et Technique) et à la stratégie numérique.
- **Portfolio Interactif :** Vitrine des projets réalisés (Canal+, Kepler, GIZ, ATR, Aldelia, etc.).
- **Blog & Ressources :** Articles sur les dernières tendances technologiques.
- **Approche Client :** Détails sur la planification stratégique et l'exécution innovante.

## 🛠️ Technologies & Expertise

Le projet met en avant une large gamme de technologies maîtrisées :

- **Frontend :** ReactJS, Vue.js, Angular, Next.js.
- **Backend :** Node.js, Python (Django), PHP (Laravel), Ruby on Rails, .NET, Java.
- **Cloud & DevOps :** AWS, Google Cloud, Azure, Kubernetes, Jenkins, Firebase.
- **Bases de données :** MySQL, MongoDB, Redis, Apache.

## 📂 Structure du Projet (Build)

Ce répertoire contient la version de production optimisée de l'application :

- `static/js/` : Fichiers JavaScript compilés et minifiés.
- `static/css/` : Feuilles de style CSS optimisées.
- `static/media/` : Ressources multimédias (images, icônes SVG, vidéos).
- `index.html` : Point d'entrée principal de l'application.

## 💻 Installation et Déploiement

Comme ce répertoire contient une version de production (`build`), il peut être servi directement par n'importe quel serveur web (Apache, Nginx, ou via un serveur statique Node.js).

### Serveur Local (WAMP/XAMPP)
Placez le dossier `wouessi` dans votre répertoire `www` (WAMP) ou `htdocs` (XAMPP). L'application sera accessible via `http://localhost/wouessi`.

### Utilisation de 'serve' (Node.js)
Si vous avez Node.js installé, vous pouvez servir l'application rapidement :
```bash
npx serve -s .
```
